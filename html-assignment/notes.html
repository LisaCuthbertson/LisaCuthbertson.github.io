HTML Codes

Italic - (em)
Changes desired content into Italic.
Bold - (strong)
Changes desired content to a darker tone or in other words bold.
Footer - (footer)
Adds specified content, often copyright information to the bottom of a web page. Refer to bottom of the page for an example.
Paragraph - (p)
Creates a paragraph for a web page.
Ordered list - (od)
Creates an ordered numbered list of specified elements or components.
